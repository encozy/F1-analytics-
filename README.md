# 🏎️ F1 Race Lab

Интерактивный анализ **любой сессии Формулы-1 с 2018 года** на реальных данных официального тайминга (библиотека FastF1): гоночный темп, стинты и деградация резины, телеметрия, квалификация.

## Запуск (macOS)

```bash
cd f1lab
chmod +x run.sh
./run.sh
```

Скрипт сам создаст окружение, поставит зависимости и откроет приложение в браузере (обычно http://localhost:8501).

Вручную то же самое:

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

Если система ругается на версию Python — нужен Python 3.10+: `brew install python@3.12`.

## Как пользоваться

1. Слева выбери сезон → Гран-при → сессию (гонка, квалификация, спринт, практики) → «Загрузить сессию».
2. Первая загрузка сессии занимает 1–3 минуты — качаются официальные тайминги и телеметрия. Всё кэшируется в `./cache`, повторное открытие мгновенное.
3. Вкладки:
   - **Темп** — боксплоты времени круга по пилотам + race trace хода гонки;
   - **Шины и стинты** — лента стратегий и график деградации с расчётом темпа износа (с/круг), поправка на топливо настраивается слайдером;
   - **Телеметрия** — скорость по дистанции для пары пилотов с отметками поворотов, дельта времени, speed trap, аэро-скэттер по командам;
   - **Квалификация** — отставания от поула, лучшие сектора, теоретический круг;
   - **Итоги** — официальный протокол.

## Стек

Python · FastF1 · pandas · Plotly · Streamlit. Данные: официальный тайминг F1 (через FastF1). Проект учебно-аналитический, не аффилирован с Formula 1.
